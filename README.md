## ✨ 機能

### 主要機能

1. **インタラクティブマップ**
   - Google Maps APIを使用した高性能な地図表示
   - ダークモードデザインで心霊スポットの雰囲気を演出
   - 18箇所の心霊スポットをマーカーで表示

2. **スポット情報**
   - ランキング形式での表示（第1位〜第18位）
   - 各スポットの詳細情報（住所、説明、危険度）
   - スポットタイプ別のアイコン表示

3. **検索・フィルター**
   - リアルタイム検索機能
   - タイプ別フィルター（廃墟、橋、トンネル、岬、神社、公園、海岸）
   - マーカーとリストの連動表示

4. **レスポンシブデザイン**
   - デスクトップ、タブレット、スマートフォンに対応
   - モバイルではサイドバーを折りたたみ可能

5. **詳細モーダル**
   - 各スポットの詳細情報をモーダルで表示
   - 危険度表示（5段階評価）
   - 警告メッセージ表示

## 🚀 セットアップ

### 必要要件

- Google Maps API キー
- モダンブラウザ（Chrome, Firefox, Safari, Edge）
- インターネット接続

### Google Maps API キーの取得

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセス
2. 新しいプロジェクトを作成
3. 「APIとサービス」→「ライブラリ」から「Maps JavaScript API」を有効化
4. 「認証情報」からAPIキーを作成
5. 必要に応じてAPIキーの使用制限を設定

### インストール手順

1. **ファイルのダウンロード**
   ```bash
   # すべてのファイルを同じディレクトリに配置
   haunted-spots/
   ├── index.html
   ├── style.css
   ├── app.js
   └── README.md
   ```

2. **APIキーの設定**
   
   `index.html` の最後にある以下の行を編集:
   ```html
   <script async defer 
       src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY_HERE&callback=initMap&language=ja">
   </script>
   ```
   
   `YOUR_API_KEY_HERE` を取得したAPIキーに置き換え:
   ```html
   <script async defer 
       src="https://maps.googleapis.com/maps/api/js?key=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXX&callback=initMap&language=ja">
   </script>
   ```

3. **起動**
   
   ブラウザで `index.html` を開く、またはローカルサーバーを起動:
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsのhttpサーバーの場合
   npx http-server
   ```
   
   その後、ブラウザで `http://localhost:8000` にアクセス

## 📖 使用方法

### 基本操作

1. **スポットの選択**
   - サイドバーのリストをクリック
   - マップ上のマーカーをクリック

2. **検索**
   - 上部の検索ボックスにスポット名を入力
   - リアルタイムで結果が絞り込まれます

3. **フィルター**
   - タイプ選択ドロップダウンで特定のタイプのみ表示
   - 廃墟、橋、トンネルなどカテゴリ別に絞り込み

4. **詳細情報の表示**
   - マーカーをクリックして情報ウィンドウを表示
   - 「詳細を見る」ボタンでモーダル表示

5. **マップのリセット**
   - 「🔄 リセット」ボタンで初期状態に戻る

6. **モバイル操作**
   - 「☰ メニュー」ボタンでサイドバーを表示/非表示

## ⚠️ 注意事項

### 重要な警告

1. **実際の訪問は危険です**
   - このアプリは情報提供のみを目的としています
   - 実際に訪問することは推奨しません

2. **法律遵守**
   - 私有地への無断侵入は不法侵入罪になります
   - 廃墟への侵入は建造物侵入罪に該当する可能性があります

3. **安全第一**
   - 廃墟は崩落の危険があります
   - 夜間の訪問は特に危険です
   - 単独での行動は避けてください

4. **データについて**
   - スポット情報はインターネット上の情報を基にしています
   - 座標は概算であり、正確な位置とは限りません
   - 現状と異なる場合があります

### APIキーの保護

- **本番環境では必ずAPIキーの制限を設定してください**
  - HTTPリファラーの制限
  - IPアドレスの制限
  - API使用量の監視

- Firefox 88+
- Safari 14+
- Edge 90+

## 🔧 カスタマイズ

### スポットの追加

`app.js` の `hauntedSpots` 配列に新しいオブジェクトを追加:

```javascript
{
    rank: 19,
    name: "新しいスポット",
    lat: 33.5597,
    lng: 133.5311,
    address: "住所",
    description: "説明",
    type: "廃墟",
    details: "詳細情報",
    danger: 3
}
```

### カラーテーマの変更

`style.css` の CSS変数を編集:

```css
:root {
    --primary-color: #ff6b6b;  /* メインカラー */
    --secondary-color: #1e1e2e; /* 背景色 */
    --accent-color: #2d2d44;    /* アクセントカラー */
}
```
